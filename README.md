# 🛒 Super Store Sales & Scenario Dashboard — Power BI

A multi-page interactive Power BI report built on the Super Store dataset, covering sales performance, product analysis, scenario modelling, and root-cause analysis. Designed for executive-level exploration with full cross-page interactivity and custom DAX measures.

---

## 📊 Dashboard Pages

| Page | Description |
|---|---|
| **Cover** | Navigation landing page with quick access to all report sections |
| **Sales Overview** | Core KPIs (Total Sales, Quantity, Profit) with time-series and category breakdowns |
| **Product Analysis** | Product count trends by quarter, month, and sub-category |
| **What-IF Analysis** | Scenario modelling — simulate the impact of changing price, quantity, discount, and cost simultaneously |
| **Why?** | Decomposition Tree for dynamic drill-down into sales performance drivers |
| **Tooltips** | Custom YOY gauge tooltips embedded across Sales and Product pages |

---

## ✨ Key Features

- **What-IF Scenario Engine** — Built with DAX parameters to model real-time revenue impact across 4 variables: Price, Quantity, Discount, and Cost
- **Year-over-Year (YOY) Intelligence** — DAX time intelligence functions surfaced through custom tooltip pages with gauge visuals
- **Decomposition Tree** — Dynamic root-cause analysis of sales by category and sub-category
- **4-Dimensional Slicing** — Year, Category, Segment, and State filters applied across all pages with full cross-page filtering
- **Executive Navigation** — Custom cover page with action buttons for a clean, app-like user experience

---

## 🛠️ Tools & Technologies

- **Power BI Desktop**
- **DAX** — KPI measures, time intelligence (YOY), What-IF parameters, scenario calculations
- **Super Store Dataset** — Orders, customers, products, and regional sales data
